# Titanic-Iris-Dataset

This is a Machine Learning project using Titanic and Iris datasets.

## Features
- Data Cleaning
- Data Visualization
- Model Training
- Prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Datasets
- Titanic Dataset
- Iris Dataset

## Machine Learning Algorithms
- Logistic Regression
- Decision Tree
- KNN

## How to Run

1. Open project folder
2. Run Jupyter Notebook
3. Execute all cells
